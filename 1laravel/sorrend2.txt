5. MIGRATION SZERKESZTÉS -> s ek átnevezése névben fáljban 2 helyen
use Illuminate\Support\Facades\DB; kell ha használok idot DB:raw hoz
database/migrations/ mappában
KATEGORIAK tábla:
Schema::create('kategoriak', function (Blueprint $table) {
    $table->id();
    $table->string('nev');
});
TURAK tábla:
Schema::create('turak', function (Blueprint $table) {
    $table->id();
    $table->foreignId('kategoria_id')->constrained('kategoriak');
	$table->foreignId('alap tabla')->references('kapcsolati tábla')->on('masik tablaban a mező neve primary key-e')
	$table->date('Datuma')->nullable()->default(DB::raw('CURRENT_TIMESTAMP')); 
    $table->string('leiras')->nullable();
    $table->date('indulas')->nullable();
    $table->integer('tav');
    $table->boolean('aktiv')->default(true);
    $table->string('kep')->nullable();
});
Hasznos mezőtípusok:
->nullable()          = mező lehet üres
->default(true)       = alapértelmezett érték
->unique()            = egyedi érték
->constrained('tabla')= idegen kulcs

7. MODEL SZERKESZTÉS kapcsolatok az álján mitában van 
app/Models/ mappában
<?php
namespace App\Models;
use Illuminate\Database\Eloquent\Model;

class Turak extends Model
{
    public $timestamps = false;

    protected $fillable = [
        'kategoria_id',
        'leiras',
        'indulas',
        'tav',
        'aktiv',
        'kep',
    ];
}
FONTOS: $timestamps = false; ha nincs created_at/updated_at a táblában!
Alternatíva fillable helyett: protected $guarded = [];

9. CONTROLLER SZERKESZTÉS
app/Http/Controllers/ mappában
Fejléc (mindig kell):
use App\Models\Turak;
use App\Models\Kategoriak;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Validator;
GET – összes adat:
public function index()
{
    $turak = Turak::all();
    return response()->json($turak, 200);
}
GET – id alapján (konkrét rekord):
public function show($id)
{
    $tura = Turak::find($id);
    if (!$tura) {
        return response()->json("A tura nem letezik.", 404);
    }
    return response()->json($tura, 200);
}
GET – idegen kulcs alapján (pl. kategória összes túrája):
public function show($id)
{
    $turak = Turak::where('kategoria_id', $id)->get();
    return response()->json($turak, 200);
}
POST – új rekord (foreach validáció):
public function store(Request $request)
{
    $required = ['kategoria_id', 'leiras', 'tav', 'aktiv'];

    foreach ($required as $field) {
        if (!$request->has($field)) {
            return response()->json("Hianyos adatok.", 400);
        }
    }

    $tura = Turak::create([
        'kategoria_id' => $request->input('kategoria_id'),
        'leiras'       => $request->input('leiras'),
        'indulas'      => $request->input('indulas'),
        'tav'          => $request->input('tav'),
        'aktiv'        => $request->input('aktiv'),
        'kep'          => $request->input('kep'),
    ]);

    return response()->json(['id' => $tura->id], 201);
}
POST – új rekord (Validator verzió):
public function store(Request $request)
{
    $validator = Validator::make($request->all(), [
        'kategoria_id' => 'required',
        'tav'          => 'required',
        'aktiv'        => 'required',
    ]);

    if ($validator->fails()) {
        return response()->json("Hianyos adatok.", 400);
    }

    $tura = Turak::create($request->all());
    return response()->json(['id' => $tura->id], 201);
}
PATCH – módosítás (required nélkül):
public function update(Request $request, $id)
{
    $tura = Turak::find($id);

    if (!$tura) {
        return response()->json("A tura nem letezik.", 404);
    }

    $data = $request->only(['kategoria_id','leiras','indulas','tav','aktiv','kep']);
    $tura->fill($data);
    $tura->save();

    return response()->json($tura, 200);
}
PATCH – módosítás (required mezőkkel):
public function update(Request $request, $id)
{
    $validator = Validator::make($request->all(), [
        'kategoria_id' => 'required',
        'tav'          => 'required',
    ]);

    if ($validator->fails()) {
        return response()->json("Hianyos adatok.", 400);
    }

    $tura = Turak::find($id);

    if (!$tura) {
        return response()->json("A tura nem letezik.", 404);
    }

    $tura->update($request->all());
    return response()->json($tura, 200);
}
DELETE:
public function destroy($id)
{
    $tura = Turak::find($id);

    if (!$tura) {
        return response()->json("A tura nem letezik.", 404);
    }

    $tura->delete();
    return response()->json('', 204);
}

10. API.PHP ÚTVONALAK
routes/api.php:
<?php
use Illuminate\Support\Facades\Route;
use App\Http\Controllers\TurakController;

Route::get('/kategoria/{id}', [TurakController::class, 'show']);
Route::post('/tura', [TurakController::class, 'store']);
Route::patch('/tura/{id}', [TurakController::class, 'update']);
Route::delete('/tura/{id}', [TurakController::class, 'destroy']);
Ha az api.php nem létezik, app.php-ba kell beírni:
->withRouting(
    web: __DIR__.'/../routes/web.php',
    api: __DIR__.'/../routes/api.php',
    commands: __DIR__.'/../routes/console.php',
    health: '/up',
)
